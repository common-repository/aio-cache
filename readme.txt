== Description ==

AIO Cache, a stable and simple cache, minify, and performance solution for those of you looking for solid raw performance and response times. __Fast, Powerful, and Effective!__

Unlike other performance solutions, you don't need to configure AIO Cache. Just install and go. __Quick and Simple!__

Best part is that it is designed to avoid breaking your site. __Stable and Solid!__

= AIO Cache Offers: =

*   Simple Page & File Caching
*   Simple Static Resources
*   Simple GZip & Deflate Compression
*   JavaScripts to Footer
